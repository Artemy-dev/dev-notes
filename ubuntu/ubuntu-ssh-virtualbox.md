# Подключение к Ubuntu Server по SSH в VirtualBox

## Настройка сети в VirtualBox

1. Откройте **VirtualBox** и выберите нужную виртуальную машину
2. Перейдите в меню **«Настройки» → «Сеть»**
3. Установите **«Тип подключения»: Сетевой мост**
4. В поле **«Имя»** выберите сетевой адаптер вашего компьютера

Режим сетевого моста позволяет виртуальной машине выглядеть в сети как отдельный физический компьютер и получать IP-адрес от роутера, что и нужно для SSH-доступа из основной системы.

## Подготовка Ubuntu Server

### Установка SSH-сервера

Обновите пакеты и установите OpenSSH-сервер:

```bash
sudo apt update 
sudo apt install openssh-server 
```

### Проверка и настройка службы SSH

Проверьте статус службы SSH:

```bash
sudo systemctl status ssh
```

Включите автоматический запуск SSH при загрузке системы:

```bash
sudo systemctl enable ssh
```

### Определение IP-адреса сервера

Получите IP-адрес сервера для подключения:

```bash
hostname -I
```

## Подключение с клиентской машины

### Использование Termius

1. Запустите приложение **Termius**
2. Создайте новое подключение: **«NEW HOST»**
3. Заполните параметры подключения:
    - **Address:** IP-адрес сервера (например, 192.168.0.108)
    - **Username:** имя пользователя сервера (например, Admin)
    - **Password:** пароль пользователя сервера (например, Admin)

Подключение через терминал:

```bash
ssh username@server_ip 
```

Пример:

```bash
ssh Admin@192.168.0.108
```

Где:

- **Admin** - имя пользователя на сервере
- **192.168.0.108** - IP-адрес сервера

После выполнения команды введите пароль пользователя при запросе.

## Рекомендации по безопасности

### Создание отдельного пользователя для SSH

Настоятельно рекомендуется создать отдельного пользователя для SSH доступа вместо использования стандартной учетной записи. Выполните команду для создания нового пользователя (замените newuser на желаемое имя):

```bash
sudo adduser newuser
```

Система запросит ввести и подтвердить пароль для нового пользователя, а также дополнительную информацию.

### Требования к надежному паролю

Используйте сложные пароли, соответствующие следующим критериям:

- Длина: не менее 12 символов
- Регистры: заглавные и строчные буквы
- Цифры: минимум одна цифра
- Специальные символы: например !@#$%^&*()

Пример надежного пароля:

```txt
Y@lT:y#GfJ*m
```

После создания пользователя можно подключиться через SSH используя новые учетные данные:

```bash
ssh newuser@192.168.0.108
```
