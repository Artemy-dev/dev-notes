## Redis - NoSQL база данных «ключ-значение»

**Redis (Remote Dictionary Server)** - это высокопроизводительное in-memory хранилище данных с поддержкой персистентности на диск. Используется как кэш, брокер сообщений (для **Celery**), база для сессий и т.д.

Официальный сайт: https://redis.io

### Установка на Windows

*Официальной сборки Redis для Windows нет. Рекомендуемый порт: https://github.com/tporadowski/redis/releases*

* Скачать ZIP-архив последнего релиза (например, `Redis-x64-5.0.14.1.zip`).
* Распаковать в удобную папку (например, `C:\Redis`).

### Запуск сервера и клиента

Запуск **сервера** (в фоне):

```
redis-server.exe
```
Сервер поднимается на `127.0.0.1:6379`.

Запуск **клиента**

```
redis-cli.exe
```

### Проверка работы в консоли redis-cli

```redis
> keys *
(empty list or set)

> set a 123
OK

> keys *
1) "a"

> get a
"123"
```

* Важно: В реальных проектах избегайте команды keys * - она блокирует сервер при большом объёме данных. Вместо неё используйте SCAN 0.*

### Кэширование в FastAPI с Redis

Для кэширования ответов эндпоинтов **FastAPI** используется библиотека `fastapi-cache2`.

Репозиторий: https://github.com/long2ice/fastapi-cache

Установка с поддержкой Redis

```
pip install "fastapi-cache2[redis]"
```
