## Настройка v2rayN

Запустить **v2rayN (Запуск от имени Администратора)**

На главном экране установить:

`Режим VPN (On)` / `Установить системный прокси` / `V4-(Whitelist)`

**Настройки -> Настройки параметров:**

* `Базовые натсройки` -> `Разрешить подключение из локальной сети (On)`
* `Настрйоки режима TUN` -> `Автроматческая маршрутизация (On)` и `Строгая маршрутизация (Off)`

**Настройки -> Настройки DNS:**

`Расширенные настройки DNS` -> `DNS hosts`:

```
8.8.8.8, 1.1.1.1
```

## Брандмауэр Windows

* `Win+R` -> `wf.msc`
* `Правила для входящих подключений`
* `Создать правило...` для C:\Program Files\v2rayN-windows-64\v2rayN.exe
* `Создать правило...` для C:\Program Files\v2rayN-windows-64\bin\xray\xray.exe

## Сетевые подключения

* Включить Мобильный хот-спот
* `Win+R` -> `ncpa.cpl`
* `singbox_tun` -> `Свойства` -> `Доступ` -> Поставить галочки и выбрать `Подключение по локальной сети* 10`
