### Celery

Библиотека для **Python**, которая позволяет выполнять задачи в фоновом режиме.
**Celery** хранит задачи в **Redis**. **Redis** используется как брокер.

**Сценарий использования**

Пример: рассылка сообщений, каждый день в 09:00 всем сотрудникам.

**Установка**

```
pip install celery flower
```

* **flower** - для трекинга задач в web-интерфейсе.

В каталоге с проектом создаем каталог `tasks/` и в нем файлы `celery.py` и `tasks.py`.

**Запуск (в корневой директории проекта)**

```
celery -A app.tasks.cel:celery worker --loglevel=INFO --pool=solo
```

* **app.tasks.cel** - путь до файла.
* **:celery** - имя переменной (объект класса **Celery**).

Если вы работаете на **Windows**, всегда указывайте `--pool=solo` при запуске **Celery**, иначе
задачи не будут выполняться. Для других ОС такого ограничения нет.
